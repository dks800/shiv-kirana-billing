## ✨ Overview

**Shiv Kariyana Billing** is a modern retail billing application designed for Kirana shops and provision stores. It simplifies invoice creation, product management, sales tracking, GST reporting, and business analytics through an intuitive and responsive user interface.

The application focuses on speed, simplicity, and accuracy while providing store owners with meaningful business insights.
